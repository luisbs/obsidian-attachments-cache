## Other filetypes

By default Obsidian only renders images, any other filetypes can be used as an attachment if the file is inside the vault. The plugin uses that to render cached attachments.

> Other supported formats include:
> - Audio: `.flac`, `.m4a`, `.mp3`, `.ogg`, `.wav`, `.webm`, `.3gp`
> - Video: `.mkv`, `.mov`, `.mp4`, `.ogv`, `.webm`
> - Documents: `.pdf`
